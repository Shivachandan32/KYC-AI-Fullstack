# KYC-AI Fullstack

AI-powered document verification system using RAG and LLMs.

## Features
- Document Upload
- AI Verification
- Compliance Rule Retrieval
- Explainable Decisions
- RAG Pipeline
- Groq Vision Integration

## Tech Stack

Backend:
- FastAPI
- Python
- ChromaDB
- Groq

Frontend:
- HTML
- CSS
- JavaScript

↓

Explanation Generation
